# TabBar 图标说明

请准备以下图标文件（建议尺寸 81x81 px，PNG 格式）：

1. `chat.png` - 对话图标（未选中状态）
2. `chat-active.png` - 对话图标（选中状态，绿色）
3. `knowledge.png` - 知识库图标（未选中状态）
4. `knowledge-active.png` - 知识库图标（选中状态，绿色）
5. `mine.png` - 我的图标（未选中状态）
6. `mine-active.png` - 我的图标（选中状态，绿色）

推荐图标资源：
- iconfont: https://www.iconfont.cn/
- iconpark: https://iconpark.oceanengine.com/

临时方案：可以先注释掉 pages.json 中的 iconPath 配置，使用纯文字 tabBar。
